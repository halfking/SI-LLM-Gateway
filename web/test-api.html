使用说明

  1. 确认 API Base URL 是否正确（默认 https://[PROD_DOMAIN]）
  2. 获取 Auth Token：
    • 方法 1: 在浏览器访问 https://[PROD_DOMAIN]，F12 打开 DevTools → Application → Cookies，找到 auth token
    • 方法 2: 使用登录 API 获取 token
  3. 填写 Auth Token 到输入框
  4. 点击"测试 /api/logs (带认证)"按钮
  5. 查看响应结果
