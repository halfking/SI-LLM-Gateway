💡 关键学习点
================================================================================

1. 视图依赖的重要性
   - v_routable_credential_models 是路由系统的核心
   - 视图不存在会导致所有节点不可用

2. SQL LEFT JOIN 和 NULL 语义
   - LEFT JOIN 未匹配返回 NULL
   - NULL = TRUE 返回 FALSE (不是 NULL)
   - 导致 WHERE 条件过滤所有记录

3. 多层状态检查
   - Provider 级别: enabled, manual_disabled
   - Credential 级别: status, lifecycle, availability, quota
   - Model Offer 级别: available, unavailable_reason
